# Weather CLI Example

This example demonstrates how to use KagentAI to build a command-line interface tool using Node.js.

## Task Description

"Create a CLI tool that fetches weather information for a given city.
- Use a public weather API (e.g., OpenMeteo)
- Accept city name as an argument
- Display temperature, humidity, and conditions
- Support unit selection (Celsius/Fahrenheit)"

## Agent Workflow

1. **Planner Agent**: Plans the CLI structure and API integration.
2. **Architect Agent**: Designs the command structure and API client.
3. **Coder Agent**: Implements the CLI using `commander` and `axios`.
4. **Reviewer Agent**: Checks for error handling and code quality.

## How to Run

1. Open KagentAI
2. Select "Start New Task"
3. Paste the task description above
4. Watch the agents build the CLI tool!
