# Todo App Example

This example demonstrates how to use KagentAI to build a simple Todo application using React and TypeScript.

## Task Description

"Create a Todo application with the following features:
- Add, edit, and delete todos
- Mark todos as complete
- Filter by status (all, active, completed)
- Persist data to local storage"

## Agent Workflow

1. **Planner Agent**: Breaks down the task into steps:
   - Setup React project
   - Create Todo types and state management
   - Build UI components
   - Implement logic
   - Add persistence

2. **Architect Agent**: Designs the component structure:
   - `App.tsx`: Main container
   - `TodoList.tsx`: List of todos
   - `TodoItem.tsx`: Individual todo item
   - `TodoFilter.tsx`: Filter controls
   - `useTodos.ts`: Custom hook for logic

3. **Coder Agent**: Generates the code for each component.

4. **Tester Agent**: Verifies functionality with unit tests.

## How to Run

1. Open KagentAI
2. Select "Start New Task"
3. Paste the task description above
4. Watch the agents build the application!
