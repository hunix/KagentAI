# Blog API Example

This example demonstrates how to use KagentAI to build a RESTful API using Express and TypeScript.

## Task Description

"Create a REST API for a blog platform.
- Endpoints for Posts (CRUD)
- Endpoints for Comments
- In-memory storage (for simplicity)
- Input validation
- Error handling"

## Agent Workflow

1. **Planner Agent**: Outlines the API endpoints and data models.
2. **Architect Agent**: Designs the Express app structure (routes, controllers, services).
3. **Coder Agent**: Implements the API with validation and error handling.
4. **Tester Agent**: Generates integration tests for the endpoints.

## How to Run

1. Open KagentAI
2. Select "Start New Task"
3. Paste the task description above
4. Watch the agents build the API!
